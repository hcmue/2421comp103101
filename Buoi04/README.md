# React Router

`https://reactrouter.com/`

## Demo

- Tạo project dùng vite template react (jsx)
  `npm create vite@latest demorouter -- --template react`

- Dựng lại thư viện: `npm i`

- Cài thư viện react router: `npm i react-router-dom`
